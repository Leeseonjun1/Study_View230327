목록을 만들때 사용하는 태그

크게 3가지 목록이 있고, 기본적인 문법과 사용방법 익히기

순서가 없는 목록

Unordered List를 줄여서 ul이라고 하는 명령어로 표현하고, 내부항목은 List Item 이라는 의미를 가진 li 태그명령

내가 좋아하는 것(음식)

  • 김치
  • 피자
  • 삼겹살

순서가 있는 목록

Ordered List를 줄여서 ol 이라고 하는 명령어로 표현

올해 가고싶은 여행지 (순서)

  1. 베트남
  2. 필리핀
  3. 괌

사용자 정의 목록

Definition List를 줄여서 dl 이라는 태그로 표현하고, 내부항목은 용어를 뜻하는 Definition Term,dt태그 그리고 내용은 Definition Description 즉, dd태그로 표현한다

자동차

비행기
이동수단,운송수단으로 하늘을 나는 기계장치
스쿠터
새롭게 등장한 이동수단, 면허가 없어도 이용할수 있고 시간당 1000원의 비용이 든다

  3. 물이 끓으면 스프를 넣는다
  4. 기다린다
  5. 맛있게먹는다
